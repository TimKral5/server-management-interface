# Styleguide

## Typography

 - Font Family: `Arial, Helvetica, sans-serif`
 - Font Sizes:
    - Heading 1: `2rem`
    - Heading 2: `1.8rem`
    - Heading 3: `1.6rem`
    - Heading 4: `1.4rem`
    - Heading 5: `1.2rem`
    - Heading 6: `1rem`
