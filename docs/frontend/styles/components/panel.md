# Panel

## Usage
```html
<!-- With text as Content -->
<div class="awi-panel">
   <span class="title"></span>
   <p>
      Content
   </p>
</div>

<!-- With table as Content -->
<div class="awi-panel">
   <span class="title"></span>
   <table>
      <tbody>
         <tr>
            <th>Head 1</th>
            <th>Head 2</th>
         <tr>
         <tr>
            <td>Col 1</td>
            <td>Col 2</td>
         </tr>
      </tbody>
   </table>
</div>
```
