# General Layout

The layout is enabled by adding the class `awi-layout` to the body
tag.

## Usage

```html
<html>
<head>
   ...
</head>
<body class="awi-layout">
   <header>
      <div class="logo"><a href="...">Site</a></div>
   </header>
   <nav>
      <li><a href="...">Page 1</a></li>
      <li><a href="...">Page 2</a></li>
      <li><a href="...">Page 3</a></li>
   </nav>
   <article>
      <h1>Page X</h1>
      Lorem Ipsum...
   </article>
   <footer>
      <a href="...">Help?</a>
   </footer>
</body>
</html>
```
